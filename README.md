# TestCalculator
A simple score calculator of your examinations, provided in Finnish Grading Sytem.

## Description
It is a simple calculator of your score throughout examinations, with the addition of providing Finnish Grades. It is quite harsh for lower grades, but generous for the higher grades.

## Requirements
- Python 3.12
- customtkinter
- tkinter

## Command to install all libraries
``pip install customtkinter``
